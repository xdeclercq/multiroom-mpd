Resource 'zone'

A zone.

Properties

Name Description Type
name The zone name String

Links

Relation Description Methods Templated
self The zone GET No
mr:player A link to the player resource of this zone GET No

Example

{
  "_links": {
    "self": {
      "href": "http://localhost:8080/multiroom-mpd/api/zones/Kitchen"
    },
    "mr:player": {
      "href": "http://localhost:8080/multiroom-mpd/api/zones/Kitchen/player"
    }
  },
  "name": "Kitchen"
}
