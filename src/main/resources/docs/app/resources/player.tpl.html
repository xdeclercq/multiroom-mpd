Resource 'player'

A player of a zone.

Properties

Name Description Type
name The zone name String

Links

Relation Description Methods Templated
self The player GET No
mr:play A link to play music and retrieve the player resource POST No
mr:pause A link to put the music on hold and retrieve the player resource POST No
mr:stop A link to stop the player and retrieve the player resource POST No
mr:status A link to the status resource of this player WS No

Embedded

Relation Description Methods Templated
mr:status The status resource of this player WS No
